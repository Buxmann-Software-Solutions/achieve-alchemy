
## Getting Started

To get started with the project, follow these steps:

1. Install the dependencies:
   ```sh
   bun install
   ```

2. Start the development server:
   ```sh
   bun run tauri dev
   ```

3. Build the app:
   ```sh
   bun run tauri build
   ```

   ## Built With

   - [Tauri](https://tauri.app/)
   - [Diesel](https://diesel.rs/)
   - [React](https://reactjs.org/)
